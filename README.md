# BoardGameCafe
this is university mini project

##프로젝트 내용
 BoardGameCafeManager project는 보드게임카페에서 사용할 수 있는 관리 프로그램입니다.

 ##기능
 1. 회원 관리기능(회원 등록, 회원 검색, 회원 탈퇴 또는 삭제등)
 2. 자리 관리기능(자리 인원수,자리를 이용하는 사용자 정보, 자리유무확인등)
 3. 통계 기능(회원의 방문 빈도수, 이용시간등)

 ##작성 규칙
 1. 모든 함수나 변수가 추가될때에는 주석으로 설명을 달아놓을 것.
 2. main function에 변수 추가또는 삭제를 해야될시, 적절한 의견교환을 통해 추가, 삭제를 할 수 있음.
 3. main function은 특수한 상황이 아니면 절대 개인적인 수정은 금할 것(주석은 허용).
 4. 규칙은 작성도중 문제 발생시 적절한 의견교환으로 규칙을 수정또는 추가할 수 있음.
